### 분석 지시사항

1. **프로필 강약점 분석** (`profileAnalysis`)
   - 지원자의 보유 스킬과 Top 3 공고의 요구 스킬을 교차 분석하여 강점과 보완점을 도출하세요.
   - 강점: 공고 요구사항과 잘 맞는 부분, 어필 포인트
   - 약점: 공고 요구 대비 부족한 스킬·경험, 보완 우선순위 순으로 정렬
   - fitScore: 1순위 공고 기준 종합 매칭 평가 (한 줄 요약)

2. **이력서 개선** (`resumeImprovements`, 3~5개 항목)
   - 게임 업계 채용 기준에서 이력서 구성·내용의 개선 포인트를 제시하세요.
   - 경력자라면 성과 수치 중심, 신입이라면 기술 깊이와 기여도 중심으로 작성하세요.

3. **자기소개서 개선** (`coverLetterImprovements`)
   - `common` (2~3개): 어느 공고에나 공통으로 적용할 수 있는 자소서 개선 포인트. 지원 동기의 구체성, 직무 이해도 어필 방식 등.
   - `rank1` (2~3개): 1순위 공고의 인재상·기술 키워드·최신 이슈와 자소서를 직접 연결하는 맞춤 전략.
   - `rank2` (2~3개): 2순위 공고 기준으로 자소서에서 강조해야 할 차별화 포인트.
   - `rank3` (2~3개): 3순위 공고 기준으로 자소서에서 강조해야 할 차별화 포인트.
   - 각 항목은 두괄식으로, 해당 공고명을 언급하여 어느 공고에 대한 피드백인지 명확히 하세요.

4. **포트폴리오 개선** (`portfolioImprovements`, 3~5개 항목)
   - {{PORTFOLIO_INSTRUCTION}}
   - 게임 업계에서 포트폴리오를 보는 관점(기여도·과정·결과 지표)을 반영하세요.

5. **면접 준비** (`interviewPreps`)
   - Top 3 공고 각각에 대해 면접 질문 3~4개를 생성하세요.
   - 질문 유형(type)은 반드시 아래 중 하나로 분류하세요:
     - `직무역량`: 기술 스택, 실무 경험 검증
     - `인성·문화핏`: 협업 방식, 갈등 해결, 조직 적응력
     - `산업이해`: 게임 시장·트렌드·해당 회사 이해도
     - `문제해결`: 실제 문제 상황 대처 방식 (STAR 기법 유도)
   - 각 질문마다 피해야 할 답변 패턴(`avoid`)과 권장 답변 방향(`recommend`)을 함께 제시하세요.
